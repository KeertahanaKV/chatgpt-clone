# ChatGPT Clone

A ChatGPT UI clone built using React and Material UI.

---

## Features

- Sidebar with collapse animation
- New Chat functionality
- Chat auto-creation
- User and assistant message bubbles
- Clean dark UI same as chatgpt

---

## Tech Stack

- React
- Vite
- Material UI
